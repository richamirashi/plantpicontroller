# plantpicontroller
Raspberry pi controller for smart watering system

## Raspberry pi setup

### Install

```
pip install AWSIoTPythonSDK
pip install boto
pip3 install adafruit-circuitpython-ads1x15
```

### Enable I2c

https://learn.sparkfun.com/tutorials/raspberry-pi-spi-and-i2c-tutorial/all
